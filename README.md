# Python Fundamentals Practice

This repository contains hands-on Python practice focused on building a strong foundation in core Python programming concepts. It is designed for beginners and anyone revising Python fundamentals for interviews, data engineering, or software development roles.

## Topics Covered

- **Variables** – Declaring and updating variables
- **Data Types** – Integers, floats, strings, booleans
- **Operators** – Arithmetic, assignment, comparison, logical, and bitwise operators
- **Lists** – Creating, indexing, slicing, and modifying lists
- **Dictionaries** – Key-value pairs, accessing values, and updates
- **Lambda Functions** – Writing concise anonymous functions
- **Conditional Statements** – `if`, `else`, and `elif` logic

## Purpose

The goal of this project is to:
- Strengthen core Python concepts
- Practice writing clean and readable code
- Build confidence before moving to advanced topics like Data Structures, OOP, PySpark, and Data Engineering workflows

## Who This Is For

- Python beginners
- Data engineering or data science learners
- Anyone preparing for technical interviews
- Developers revisiting Python basics
